原文： 【双语】OpenAI发布智能体API，开启2025智能体元年 🎯 新工具与API发布 OpenAI发布了三个新工具： Web搜索工具 、 文件搜索工具 和 计算机使用工具 。这些工具旨在帮助开发者构建更可靠和实用的代理系统。Web搜索工具允许模型访问互联网信息，确保响应内容的最新性和准确性。文件搜索工具新增了元数据过滤和直接搜索端点功能，方便开发者更高效地处理私有数据。计算机使用工具则允许开发者控制计算机，自动化任务。 🚀 Responses API OpenAI推出了 Responses API ，这是一个支持多模态、多工具调用的新API。它允许开发者在一次API调用中完成多个任务，如文件搜索和Web搜索，并返回最终答案。Responses API是Chat Completions API的超集，未来将支持更多功能。 🧩 Agents SDK OpenAI还发布了 Agents SDK ，这是一个用于构建复杂代理系统的开源框架。它支持多代理协作和任务切换，开发者可以通过Python函数定义工具，并自动生成JSON模式供模型使用。Agents SDK还提供了内置的监控和追踪功能，方便开发者调试和优化代理系统。 📅 未来计划 OpenAI计划在2026年逐步淘汰Assistants API，并鼓励开发者迁移到Responses API。未来，OpenAI将继续推出更多智能模型和工具，帮助开发者构建更强大的代理系统。
