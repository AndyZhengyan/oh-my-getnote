Like Claude Code. 📚 项目概述 项目定位 ：独立教育项目，旨在通过从零开始构建AI代理，揭秘Claude Code等现代AI编码代理的工作原理。 核心目标 ：提供5个版本的渐进式实现（v0-v4），共约1100行代码，每个版本新增关键概念，帮助开发者理解AI代理的核心机制。 法律声明 ：明确与Anthropic无关联，&rdquo;Claude Code&rdquo;为Anthropic商标。 🔄 版本演进与核心特性 版本 代码量 新增功能 核心洞察 v0 ~50行 1个Bash工具，递归子代理 Bash是基础工具，核心逻辑可极度简化 v1 ~200行 4个核心工具（bash/read/write/edit） 模型即代理（Model as Agent），实现完整代理循环 v2 ~300行 任务跟踪（TodoManager） 显式规划提升复杂任务处理能力 v3 ~450行 子代理机制（Task工具+代理注册表） 分而治之（Divide and conquer），保持上下文清洁 v4 ~550行 技能机制（Skill工具+SkillLoader） 领域专业知识按需加载，知识作为一等公民 ⚙️ 核心工作模式 AI代理的本质是 工具调用循环 ，伪代码如下： while True: response = model(messages, tools) # 模型处理消息和工具 if response.stop_reason != "tool_use": # 若无需工具调用 return response.text # 返回结果 results = execute(response.tool_calls) # 执行工具调用 messages.append(results) # 将结果加入上下文 核心哲学 ：模型占80%，代码仅占20%。现代AI代理的能力源于模型的代理训练，代码的作用是提供工具并减少干预。 📂 文件结构 learn-claude-code/ ├── v0_bash_agent.py #
