原文： Qwen-Next-80B-A3B 即将发布，下一代超稀疏MoE，HF的PR泄露信息窥探一二 🔍 核心定位 Qwen3-Next系列是阿里Qwen团队推出的下一代基础模型，专为 极致上下文长度 和 大规模参数效率 优化，通过架构创新实现性能最大化与计算成本最小化。 📊 关键模型性能对比 | 模型 | 总参数 | 激活参数 | 性能表现 | 32K+上下文吞吐量提升 | |&mdash;&mdash;&mdash;&mdash;&mdash;&mdash;&ndash;|&mdash;&mdash;&ndash;|&mdash;&mdash;&mdash;-|&mdash;&mdash;&mdash;&mdash;&mdash;&mdash;&mdash;&mdash;-|&mdash;&mdash;&mdash;&mdash;&mdash;&mdash;&mdash;-| | Qwen3-Next-80B-A3B | 800亿 | 30亿 | 超越320亿参数稠密模型 | 10倍以上（对比Qwen3-32B） | | Qwen3-32B | 320亿 | 320亿 | 传统稠密模型基准 | - | 🏗️ 架构革新三大突破 1. 激进稀疏MoE架构（1:50激活率） 设计细节 ：512个专家（ num_experts=512 ），每个token激活10个专家（ num_experts_per_tok=10 ），实现10/512≈1:50的极低激活率 优势 ：大幅降低每token计算量（FLOPs），同时保留模型容量 2. 混合注意力机制（Hybrid Attention） 组成 ： Gated Attention （ full_attention ）：精准捕捉关键信息 Gated DeltaNet （ linear_attention ）：基于SSM架构，线性复杂度处理长依赖 层排列规则 ：每4层为一周期，3层 linear_attention + 1层 full_attention （代码验证： layer_types 配置默认按此周期排列） 3. 多令牌预测（MTP） 技术特点 ：预训练时并行预测多个未来词元，替代传统&rdquo;Next Token Prediction&rdquo; 价值 ：提升训练效率与生成文本连贯性，优化语言规划能力 💻 代码核心配置（来自PR文件） # MoE参数配置（configuration_qwen3_next.py） class Qwen3NextConfig(PretrainedConfig): def __init__( self, num_experts=512, # 总专家数 num_experts_per_tok=10, # 每token激活专家数 layer_types=None, # 混合注意力层类型 # ... ): if self.layer_types is None: # 每4层周期：3个linear_attention + 1个full_attention self.layer_types = [ "linear_attention" if bool((i + 1) % 4) else "full_attention" for i in range(self.num_hidden_layers) ]
