LLM中的熵是如何计算的？ 2.1 基石：对数概率（logprob） 模型预测时为每个词分配概率，因原始概率值过小（如0.0001），通常使用对数概率（正相关于原始概率）： 核心关系 ： 对数概率越大（越接近0），概率越高，模型越确定。 对数概率 对应概率 直观解释 0 exp(0)=1.0 100%确定 -0.5 exp(-0.5)=0.607 比较确定 -1.0 exp(-1.0)=0.368 有些不确定 -2.0 exp(-2.0)=0.135 很不确定 -10.0 exp(-10.0)=0.000045 几乎排除 -∞ exp(-∞)=0.0 完全不可能 2.2 Token熵：单个词的确定性 计算公式 ： H = -Σ (p_i * log(p_i)) - 高熵 ：模型“举棋不定”，输出分布平滑（如候选词“处理”“从”“抽取”概率接近，熵=0.8588） - 低熵 ：模型“胸有成竹”，输出分布尖锐（如候选词“字段”概率93.71%，熵=0.2348） 2.3 序列平均熵：整段文本的总体表现 计算方法 ：将所有Token的熵求平均值，反映模型生成整段文本的总体确定性。 示例 ：某序列熵值列表 [0.4861, 0.0, 0.3147, 0.0, 1.1036, 0.4861, 0.6656, 0.8533] ，平均熵=0.4887（整体确定性高）。 📈 3. 如何解读熵值？一份实用指南 经验性熵值解读标准 （单位：nats/token）： 熵值范围 含义解读 适用场景 < 0.5 非常确定 重复模式、简单续写、事实性内容 0.5 - 1.5 确定性较高 事实性内容、逻辑严谨说明文 1.5 - 2.5 黄金平衡区 兼具确定性与创造性的连贯文本 2.5 - 4.0 高创造性/不确定性 诗歌、故事（可能不连贯） > 4.0 警告区 模型困惑，生成质量可能有问题 模型大小影响 ： - 小模型(7B以下)：熵值通常偏高 (1.5-4.0 nats) - 大模型(30B以上)：熵值通常较低 (0.5-2.5 nats) 💻 4. 实战：使用vLLM计算熵 4.1 理解vLLM的Logprob输出 通过配置采样参数，让模型返回Top 10候选词及其对数概率： from vllm import SamplingParams, LLM sampling_params = SamplingParams(n=1, logprobs=10) # 返回Top 10 logprob model = LLM(model=model_path) preds = model.generate(input_list, sampling_params=sampling_params) 输出数据结构 ：包含每个Token的候选词、对数概率、排名等信息。 4.2 核心计算代码 def calculate_token_entropy(logprobs_data): token_entropies = [] for token_dict in logprobs_data: logprobs = [lp.logprob for lp in token_dict.values() if not math.isinf(lp.logprob)] if not logprobs: token_entropies.append(0.0) continue probs = np.exp(np.array(logprobs)) probs = probs / np.sum(probs) # 归一化 logprobs_normalized = np.log(probs) entropy = -np.sum(probs * logprobs_normalized) token_entropies.append(round(entropy, 4)) return token_entropies def calculate_sequence_entropy(token_entropies): total_entropy = sum(token_entropies) avg_entropy = total_entropy / len(token_entropies) return avg_entropy, total_entropy 4.3 在主循环中调用 sampling_params = SamplingParams(n=1, logprobs=10) model = LLM(model=model_path) preds = model.generate(input_list, sampling_params=sampling_params) for output_request in preds: for output in output_request.outputs: logprobs_data = output.logprobs token_entropies = calculate_token_entropy(logprobs_data) avg_entropy, total_entropy = calculate_sequence_entropy(token_entropies) 🎯 5. 总结 熵作为衡量LLM不确定性的核心指标，其价值体现在： -
